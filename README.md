# CROUP-backEnd

## 환경 셋팅 방법

해당 레퍼지토리 복제

```shell
git clone https://github.com/programmers-Y2J/CROUP-backEnd.git
```

중속성 설치

```shell
npm ci
```

## commit, PR 템플릿 적용

```shell
git commit
```

명령 이후 I 키 입력 INSERT 모드로 템플릿 작성(간단)

```shell
git push oringin feature/{name}
```

후 PR템플릿 작성(상세)

## 명령어

개발용 명령어
nodemon 적용으로 변경된 파일이 생기면 자동으로 서버 재시작

```shell
npm run dev
```

빌드 명령어

```shell
npm run build
```

빌드후 빌드 파일 실행 명령어

```shell
npm run start
```
